>## 🐼サイト概要🌹
リースやブーケ、アクセサリーやキャンドルなどのハンドメイドのお店*PandaRoses*のSNSサイトです。  
*PandaRoses* 〜名前の由来は、娘がパンダとバラが大好きだからです〜  

※この店舗は実在しませんのでご了承ください。  


>### 🐼テーマを選んだ理由🌹
幼馴染がハンドメイドをしており、  
「自分の作った商品を紹介できる場が欲しい」  
「自分の作ったものがどんな風に使ってもらえているか知りたい」    
「ハンドメイド仲間を増やしたい」    
と話していたので、希望を叶えてあげたいなと思いこのテーマにしました。  
沢山の方にハンドメイドの魅力を広められるきっかけになれば嬉しいです。  



>### 🐼ターゲットユーザ🌹
- 性別、年齢不問
- ハンドメイドに興味がある方

>### 🐼主な利用シーン🌹
- *PandaRoses*の新作紹介、閲覧。
- *PandaRoses*の商品や好きなものを投稿する時。

>### 🐼サイトURL🌹
会員側　　http://pandaroses.com/  
管理者側　http://pandaroses.com/admin/sign_in

会員側の閲覧に関しましては、URLに遷移後のトップページ上の『ゲストログイン』を  
クリックしていただくと簡単ログインができます。

>## 🐼実装機能リスト🌹
https://docs.google.com/spreadsheets/d/1TrsE2YJyeR5ztQ1w0qjBtdpUOImQ1Zk1zStROd1lJTU/edit?usp=sharing  
##### 会員側  
- 新規登録/ログイン/ログアウト/退会（論理削除）
- Topページ/商品一覧/商品詳細
- 会員情報詳細/会員情報編集
- 投稿（公開、非公開設定可）/投稿一覧（いいねランキング順）/投稿詳細/投稿編集
- コメント投稿/コメント削除
- いいね/いいね一覧
- フォロー/フォロワー一覧
- 閲覧数表示（閲覧した会員の数） 
- 会員、投稿のキーワード検索  
※商品のキーワード検索は実装しておりません  
（マルシェでの販売のお知らせなどができるようカレンダー機能や地図機能を検討中のため）


##### 管理者側
- ログイン/ログアウト
- 商品情報登録/商品詳細/商品編集/削除
- 会員情報一覧/会員情報編集
- 投稿一覧/投稿編集/削除
- コメント一覧/削除
- フォロー/フォロワー一覧
- 会員、投稿のキーワード検索
