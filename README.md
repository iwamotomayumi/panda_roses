>## 🐼サイト概要🌹
リースやブーケ、アクセサリーやキャンドルなどのハンドメイドのお店*PandaRoses*のSNSサイトです。  
*PandaRoses* 〜名前の由来は、娘がパンダとバラが大好きだからです〜  

※この店舗は実在しませんのでご了承ください。  


>### 🐼テーマを選んだ理由🌹
*PandaRoses*の商品を発信するだけでなく、  
*PandaRoses* 仲間と繋がりたい！  
*PandaRoses*の商品を通じて交流の場ができたら良いなと思いこのテーマにしました。  
沢山の方にハンドメイドの魅力を広められるきっかけになれば嬉しいです。  



>### 🐼ターゲットユーザ🌹
- 性別、年齢不問
- ハンドメイドに興味がある方

>### 🐼主な利用シーン🌹
- panda_rosesの新作紹介、閲覧。
- ハンドメイド作品の投稿などをする時。

>### 🐼サイトURL🌹
http://pandaroses.com/  

作品の閲覧に関しましては、URLに遷移後のトップページ上の『ゲストログイン』をクリックしていただくと簡単ログインができます。

>## 🐼実装機能リスト🌹

##### 会員側  
- 新規登録/ログイン/ログアウト/退会（論理削除）
- Topページ/商品一覧/商品詳細
- 会員情報詳細/会員情報編集
- 投稿（公開、非公開設定可）/投稿一覧（いいねランキング順）/投稿詳細/投稿編集
- コメント投稿/コメント削除
- いいね/いいね一覧
- フォロー/フォロワー一覧
- 閲覧数表示（閲覧した会員の数） 
- 会員、投稿のキーワード検索  
※商品のキーワード検索は実装しておりません  
（マルシェでの販売のお知らせなどができるようカレンダー機能や地図機能を検討中のため）
